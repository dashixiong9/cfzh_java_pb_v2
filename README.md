#### 软件架构
springboot2 + dubbo2.7.3, 数据库是mysql，你跑100个车场没问题，这套代码最高峰时期流水每天300多万人民币都没啥问题。
说点遗憾吧！由于忙于其它事情了，没有时间迭代，原来为了追求功能丰富，使用了大量联表，导致没法处理几百亿那种数据体量，不过根据我的经验，已经满足国内99%停车项目需求了，很多公司没有机会撑爆当前这个系统的。
兼容 臻识/华夏/芊熠/海康/大华  其它相机你们自己接入即可，很简单。网络协议支持：mqtt协议，tcp协议，http协议，websocket协议

4.  安装mysql8.0

6.  安装redis
